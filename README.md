# Cisco-Packet-Tracer---Project-Management
# _Final Smart IoT Technician Bootcamp proejct showcasing network topology design, IPv4 addressing, DHCP configuration and network testing_

## Overview

This repository contains my final project completed as part of the Smart IoT Technician Bootcamp.

The project demonstrates the design, configuration, testing, and troubleshooting of a small business/home IoT network using Cisco Packet Tracer. It showcases networking fundamentals, IPv4 addressing, DHCP configuration, wireless networking, and connectivity testing.

---

## Project Objectives

- Design a functional IoT network topology
- Configure wired and wireless devices
- Configure static and dynamic (DHCP) IPv4 addressing
- Implement secure wireless networking
- Verify network connectivity
- Troubleshoot communication issues
- Document the complete network design
